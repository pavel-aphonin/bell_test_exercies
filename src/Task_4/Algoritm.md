1. Вычисляем размерность матрицы по выбранной N: S = 2 + N * 2.
2. Строим пустую матрицу размером S на S.
3. Вычисляем индексы центральных ячеек как c1 = N и c2 = N + 1.
4. Считаем исходными ячейки (c1,c1), (c1,c2), (c2,c1), (c2,c2).
5. Для каждой ячейки вычисляем расстояния (по модулю) до обеих исходных ячеек по строке и выбираем из них меньшее.
6. Аналогично для каждой ячейки вычисляем расстояние (по модулю) до обеих исходных ячеек по колонке и берём наименьшее из них.
7. Из двух рассчитанных расстояний выбираем большее и заносим его как значение ячейки.
